# 209as-number-line

## Setup

You need Python 3.13 or higher.

Do the following

MacOS / Linux

```sh
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Windows

```sh
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
```

## Usage

Run the main simulation:

```sh
python3 main.py # Mac / Linux
python main.py # Windows
```
